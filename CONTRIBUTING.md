# Contributing

Thank you for considering a contribution to CLAIM Elaboration.

## Suggested contribution process

1. Open an issue describing the proposed correction or improvement.
2. Reference the affected CLAIM item, section, or citation.
3. Submit a pull request with a clear description of the change.
4. Keep changes focused and traceable.

## Editorial principles

- Preserve scientific meaning.
- Keep changes transparent and auditable.
- Verify references and third-party permissions before publication.
- Avoid introducing unsupported claims.
