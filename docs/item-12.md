# Item 12 — How missing data were handled.

### Clearly describe how missing data were handled. For example, describe processes to replace them with approximate, predicted, or proxy values. Discuss biases that imputed data may introduce.

Explanation and Elaboration: Missing data is an inherent challenge in real-world clinical and research datasets, often arising from incomplete patient records, technical limitations, or variations in data collection procedures. Mishandling missing values can introduce significant bias, skew algorithmic training, and ultimately lead to misguided inferences. There is no single solution to missing data. Replaced missing values can introduce bias, but excluding records with missing values may also create bias. Whatever approach is taken should be carefully considered, explained and justified.
