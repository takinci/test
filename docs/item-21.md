# Item 21 — Intended sample size

### Describe the size of the testing set and how it was determined. Use traditional power calculation methods, if applicable, to estimate the required sample size. For classification problems, in cases where there is no algorithm-specific sample size estimation method available, sample size can be estimated for a given area under the curve and confidence interval width…

Explanation and Elaboration: The size of the test set or partition should be chosen based on the acceptable level of uncertainty in performance metrics, not simply as an arbitrarily selected percentage of the number of items in the full dataset. Power calculations should be performed based on the specific performance metrics intended to be used as the primary measures or outcomes. Targeted uncertainty (e.g. confidence interval width) should be based on clinical judgment and justified.
