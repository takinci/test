# Item 38 — Estimates of diagnostic performance and their precision

### For classification tasks, include estimates of diagnostic accuracy and their measures of statistical uncertainty, such as 95% confidence intervals. Apply appropriate methodology such as receiver operating characteristic analysis and/or calibration curves. When the direct calculation of confidence intervals is not possible, report nonparametric estimates from bootstrap samples. State which variables were shown to be predictive of the response variable. Identify the subpopulation(s) for which the prediction model worked most and least effectively. If applicable, recognize the presence of class imbalance (uneven distribution across data classes within or between datasets) and provide appropriate metrics to reflect algorithm performance…

Explanation and Elaboration: Quantification of uncertainty facilitates the reader’s understanding of whether or not apparent differences in performance are significant. Raw confidence values associated with classifications produced by machine learning models are generally poor estimates of the probability that the classification is correct. Calibration maps model confidence values to calibrated probabilities so they better reflect the true likelihood that a prediction is correct [(75)](https://doi.org/10.1186/s12916-019-1466-7)[(76)](https://radiologyai.substack.com/p/why-calibration-matters-more-than). The predictive variables driving the model's performance should be explicitly stated, enabling readers to assess their clinical relevance. Subpopulation analyses are critical to identifying groups where the model performs well or poorly, ensuring transparency in its generalizability and fairness. Addressing class imbalance is particularly important in medical datasets, where uneven distributions can skew model performance [(77)](https://doi.org/10.1371/journal.pone.0293607).
